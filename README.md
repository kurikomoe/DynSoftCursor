# 竖屏鼠标指针救星（Portrait Monitor Fixer）

It is a known issue that AMD graphics drivers struggle to handle mouse movement correctly when the monitor orientation is changed. 

This application resolves this by automatically toggling between software and hardware mouse modes based on your current monitor orientation.



**Software Mouse Mode:**
Smooth but suffers from high input latency

Comparable to the feeling of using a mouse with NVIDIA DLSS Frame Generation enabled. 

Additionally, if the GPU is under heavy load, the mouse cursor may become completely unresponsive.



**Hardware Mouse Mode:**
Highly responsive and fluid, but suffers from compatibility issues with AMD drivers when the monitor is rotated.
